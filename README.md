# Single Page CV

A single page CV built with semantic HTML as part of the [roadmap.sh](https://roadmap.sh) Frontend Developer path.

## About

This project is a clean, structured CV webpage built using only HTML. The focus is on using semantic HTML elements correctly.

## Built With

- HTML

## What I Learned

- How to structure a webpage using semantic HTML tags
- Using elements like `<header>`, `<section>`, `<article>`, and `<footer>`
- How to organise content in a readable, accessible way
